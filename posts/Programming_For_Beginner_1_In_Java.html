	Using Data Type

	Variable ကြေငြာပုံ

	(Data type) (Variable Name) = (Data);

	ပထမအနေနဲ့  Data Type တွေအကြောင်းပြောပါမယ် ။

	* String

	String data type က တော့ စာကြောင်း တွေကို သိမ်းရန်ဖြစ်ပါတယ်။

	ဒီလိုသိမ်းတဲ့နေရာမှာ မိမိ သိမ်းချင်တဲ့စာသားတွေကို " နဲ့ "
	ကြားအတွင်းထည့်ပေးရပါမယ် ။

	/**************************/
	Code:
	Example :

	String name = "ZMO";
	/**************************/

	ဒီ Example မှာ name ဆိုတဲ့ variable ကို String data Type ကြေငြာလိုက်ပါတယ် ။ ပြီးတော့ "ZMO" ဆိုတဲ့ စာသားကိုသိမ်းလိုက်ပါတယ် ။

	* char

	chare data type ကတော့ စကာလုံးတစ်ခုခြင်းဆီကိုသိမ်းတဲ့နေရာမှာသုံးပါတယ် ။

	/**************************/
	Code:
	Example :

	char firstWord = 'A';
	/**************************/

	မှတ်ချက် ။ single character (သို့) စကားလုံးကိုသိမ်းတဲ့နေရာမှာ ' နဲ့ ' အတွင်းသိမ်းပေးရပါမယ်။ ( ' ) ကို  Single code လို့ ခေါ်ပြီး ( " ) ကို Double

	code လို့ခေါ်ပါတယ် ။

	* integer

	integer data type ကတော့ ကိန်းပြည့်တွေကိုသိမ်းတဲ့နေရာမှာအသုံးပြုပါတယ် ။

	/**************************/
	Code:
	Example :

	int number = 1;
	/**************************/

	မှတ်ချက် ။ ကိန်းပြည့်တွေကိုသိမ်းတဲ့အချိန်မှာ ( " ) Double code မထည့်ရပါဘူး ။ "123" နဲ့ 123 မှာ "123" ကစာသားဖြစ်ပြီး String data ဖြစ်ပါတယ် ၊

	123 ကတော့ integer data ဖြစ်ပါတယ် ။ Integer data ကသာ
	ပေါင်း + နုတ် - မြှောက် * စား / ပြုလုပ်နိုင်ပါမယ် ။

	* boolean

	boolean data type ကတော့ True အမှန် False မှား တန်ဖိုးတစ်ခုကို သိမ်းချင်တဲ့နေရာမှာအသုံးပြုပါတယ် ။

	/**************************/
	Code:
	Example :

	boolean isFinish = true;
	/**************************/

	မှတ်ချက် ။ true false တန်ဖိုးနှစ်ခုသာသိမ်းနိုင်ပါတယ် ။ true false က String မဟုတ်ပါဘူး ။ True False TRUE FALSE စသည်ရေးမရပါဘူး Error

	တက်ပါမယ် ။ true false ဆိုသည့် samll letter ဖြင့်သာရေးပါ ။

	* double

	double data type ကတော့ ဒသမကိန်းဂဏန်းတွေကို သိမ်းတဲ့နေရာမှာ အသုံးပြုပါတယ် ။ Integer မှာ သိမ်းမရပါဘူး ။

	/**************************/
	Code:
	Example :

	double rate = 1.5;
	/**************************/
